Values Join

SELECT decision.* FROM decisions d JOIN values v ON … -- not yet populated

A mindset tool for evaluating decisions: does this opportunity actually join with what matters to you, with a good use of time, and with something you can genuinely own — work on and manage — rather than something that depends on other people or circumstances? Modeled loosely on a SQL join: an opportunity only "matches" if it lines up on all three keys, not just one.

mock — the three tables
VALUES TIME WELL SPENT OWNABLE

An idea or opportunity gets entered once, then scored against each table. Where all three light up, it's a strong match — worth starting. Where one is missing, the tool names which one, instead of giving a single blended score.

  • Define your own "values" table: a short list of what actually matters, in your words
  • Define "ownable": things you can start and run yourself, without waiting on a partner or institution
  • Enter an idea or opportunity; the tool checks it against each table honestly, not to flatter it
  • Output: a match / partial-match / no-match, plus which join condition failed
  • History table: past decisions and how the joins actually played out over time

This is meant to sit alongside your coaching notes practice — a structured second opinion, not a replacement for it.
